# bilibili-m25

一个仿哔哩哔哩的本地安卓视频播放和管理应用。

## 功能特性

- 本地视频扫描与管理
- 视频播放（支持多种格式）
- 视频续播功能（记住播放位置）
- 视频收藏
- 视频搜索
- 播放历史记录
- 播放队列功能（支持上一首/下一首）
- 倍速播放功能（支持 0.5x - 2.0x）
- 主题切换（浅色/深色/跟随系统）
- 文件夹浏览模式
- 画中画模式
- 视频详情查看
- 响应式布局（支持手机、折叠屏、平板）

## 技术栈

| 类别 | 技术 |
|------|------|
| 语言 | Kotlin 2.3.20 |
| UI框架 | Jetpack Compose + Material 3 |
| 架构 | MVVM + Clean Architecture |
| 依赖注入 | Hilt |
| 本地存储 | Room + DataStore |
| 视频播放 | Media3 ExoPlayer |
| 图片加载 | Coil |
| 导航 | Navigation Compose |
| 目标SDK | Android 16 (API 36) |
